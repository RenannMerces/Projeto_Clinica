# Sistema de Clínica Médica

## Tecnologias
- Vue.js
- Node.js
- Express
- MongoDB
- Mongoose

## Como rodar o projeto

### 1 - Instalar dependências
Backend:

npm install

Frontend:

npm install

### 2 - Iniciar MongoDB

Certifique-se que o MongoDB esteja rodando localmente.

mongodb://127.0.0.1:27017/clinica

### 3 - Rodar backend

npm run dev

### 4 - Rodar frontend

npm run serve
